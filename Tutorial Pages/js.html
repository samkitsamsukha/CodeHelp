What is JavaScript?

JavaScript is a versatile programming language primarily used for creating dynamic content on webpages. Originally developed to add interactivity to web pages, it has evolved into a robust language that can run both on the client-side (in the user's browser) and server-side (using environments like Node.js).As a client-side language, JavaScript enables developers to manipulate HTML and CSS, allowing for the creation of interactive elements like forms, animations, dynamic page content updates without refreshing, and responsive user interfaces.
In a nutshell:

  • 1. Event Handling: It allows developers to respond to user actions like clicks, mouse movements, or keyboard inputs, making web pages more interactive.
  • 2. DOM Manipulation: JavaScript can access and modify the Document Object Model (DOM), enabling dynamic changes to the structure and content of a webpage after it has loaded.
  • 3. Extensibility: JavaScript can be extended through libraries and frameworks like React, Angular, and Vue.js, providing additional functionalities and simplifying complex tasks.
  • 4. Server-Side Development: With Node.js, JavaScript can run on servers, enabling developers to build server-side applications, APIs, and perform tasks traditionally done with other server-side languages.

  7. Matches, Closest, Contains 
    <body>
    <div class="box" id="id1">
        This is element 1
        <span id="sp1">Span 1</span>
    </div>
    <div class="box" id="id2">

    </div>

    let id1=document.getElementById("id1");
let sp1=document.getElementById("sp1");
console.log(id1.matches(".containers")); //false
console.log(id1.matches(".box")); //true
console.log(sp1.closest(".box")); //div.#id1.box
console.log(sp1.closest("#sp1")); //<span> id="sp1">Span 1</span>
console.log(id1.contains(sp1)); //true
console.log(sp1.contains(id1)); //false
