What is HTML?

HTML (HyperText Markup Language) was created by Tim Berners-Lee in 1991 as a standard for creating web pages. It's a markup language used to structure content on the web, defining elements like headings, paragraphs, links, and images. HTML forms the backbone of web content. In layman's terms, HTML is like the skeleton of a website. It's a set of instructions that tells a web browser how to display text, images, videos, and other elements on a webpage. Think of it as the building blocks that create the structure and look of a website, similar to how bricks and mortar are used to build a house.
In a nutshell:

  • HTML is the language of the web, used to create websites.
  • HTML defines the backbone structure or layout of web pages that we see on the Internet.
  • HTML consists of a set of tags contained within an HTML document, and the associated files typically have either a ".html" or ".htm" extension.
  • There are several versions of HTML, with HTML5 being the most recent version.

    HTML BASICS

  1. HTML Boilerplate:
    • <html> - Root of a HTML page
    • <head> - Header part of web page
    • <title> - Title of a web page
    • <body> - Body of a web page
    <html>
    <head>
        <title>Document</title>
    </head>
    <body>
        This is the body of my web page
    </body>
</html>
  2. Heading Tag: Heading tags ranging from <h1> to <h6> are used to define the structure and layout of text on a web page. These tags help create a hierarchical organization of content, making it easier for both users and search engines to understand the page's content. 
    <h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>
    Output-

    This is Heading 1

    This is Heading 2

    This is Heading 3

    This is Heading 4

    This is Heading 5
    This is Heading 6
  3. Paragraph Tag: Paragraph Tags are used to display a chunk of text items. 
    <p>This is an example of paragraph tag</p>
  4. Anchor Tag: This tag is used to attach a hyperlink to the web page. It has an attribute 'href' which is used to specify the url. 
    <a href="specified path" target="your desired location">Display Name</a>
  5. Image Tag: This tag is used to add a image to the webpage. 'src' defines the path of the image and is a mandatory attribute along with 'alt' which specifies the alternate textual content. The optional attributes include 'width' and 'height'. 
    <img src="image.png" alt="Description" width="300" height="100" />
  6. Lists: There are three types of lists in HTML
    a. Ordered List - <ol>
    b. UnOrdered List - <ul>
    c. Definition List - <dl>
    The <li> tag is used to create a List Element
    Lists also have some attributes such as type which can be used to convert bullets into square, circle or none. In case of ordered lists it can be used to convert the numbering in terms of 'I', 'i', 'a', 'A'.
    The 'start' attribute can be used define the starting point of a list and 'reversed' can be used to reverse the numbering order. 
    Ordered List
<ol type="square">
    <li>Item 1</li>
    <li>Item 2</li>
</ol>
Unordered List 
<ul type="A">
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
Definition List
<dl>
    <dt>HTML</dt>
    <dd>Hyper Text Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheet</dd>
</dl>
    Output -
    1. Item 1
    2. Item 2
    Unordered List
    • Item 1
    • Item 2
    Definition List
    HTML
    Hyper Text Markup Language
    CSS
    Cascading Style Sheet

Tables: Tables help us in data represntation and have a vast application. <table> is used to create the table, <tr> is used to create rows, <th> is used to create headings and <td> is used to enter table data. There are some other tags as well for example <caption> to put caption in the table, <thead> to specify table header, <tbody> to specify table body and <tfoot> to create table footer. The attributes rowspan & colspan are used to merge cells. 
    <table border="2px solid black">
        <tr>
            <th colspan="2">Branch Details</th>
        </tr>
        <tr>
            <th>Name</th>
            <th>Branch</th>
        </tr>
        <tr>
            <td>Person A</td>
            <td rowspan="2" align="center">CSE</td>
        </tr>
        <tr>
            <td>Person B</td>
        </tr>
    </table>
    Output -
    Branch Details
    Name Branch
    Person A CSE
    Person B

  10. Video and Audio: The <video> tag is used to embed video files in an HTML document while we use <audio> to embed audio files. It supports multiple attributes to control the video, audio playback.
      Attributes :-
    1. src - to specify path of video
    2. controls - add controls like play, pause and volume
    3. autoplay - automatically video starts when web page loads
    4. loop - repeats the video once it ends
    5. muted - mutes the video by default
     
    <video src="video.mp4" controls></video>
<audio src="audio.mp3" controls></audio>
