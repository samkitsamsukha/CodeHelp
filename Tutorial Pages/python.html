What is Python?

Python is a high-level, versatile programming language known for its simplicity, readability, and extensive community support. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability and productivity. Its syntax allows developers to express concepts in fewer lines of code than languages like C++ or Java, promoting a clean and maintainable codebase. Python is dynamically typed and supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Widely used for web development, data science, artificial intelligence, automation, and more, Python's extensive standard library and third-party packages contribute to its popularity. The language's versatility, ease of learning, and broad applicability make it a preferred choice for beginners and seasoned developers alike in a diverse range of industries.

Sequence Data Type in Python

The sequence Data Type in Python is the ordered collection of similar or different data types. Sequences allow storing of multiple values in an organized and efficient fashion. There are several sequence types in Python –


  1. Python String: In this example, we will define a string in Python and access its characters using positive and negative indexing. The 0th element will be the first character of the string whereas the - 1th element is the last character of the string. 
    String1 = "GeeksForGeeks"
print("Initial String: ")
print(String1)

# Printing First character
print("\nFirst character of String is: ")
print(String1[0])

# Printing Last character
print("\nLast character of String is: ")
print(String1[-1])
  2. Python List: 
    List = ["Geeks", "For", "Geeks"]

# accessing a element from the
# list using index number
print("Accessing a element from the list")
print(List[0])

Python Tuples: Tuples are immutable, and usually, they contain a sequence of heterogeneous elements that are accessed via unpacking or indexing (or even by attribute in the case of named tuples).
