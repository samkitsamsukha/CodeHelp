What is C++?

C++ is a powerful and versatile programming language that's often praised for its efficiency and performance. It's an extension of the C programming language and incorporates object-oriented programming (OOP) features. C++ allows developers to write code that can directly interact with hardware, making it suitable for system-level programming, such as operating systems or device drivers.
C++ is widely used in various domains such as game development, where its speed and ability to manage system resources efficiently are crucial. It's also prevalent in fields like finance, embedded systems, scientific research, and building high-performance software where control over hardware and memory is essential.The language continues to evolve with each new standard, introducing modern features while maintaining backward compatibility. C++ remains a fundamental language in the realm of programming due to its power, performance, and versatility across a wide range of applications.

  4. Statements: A statement is a basic unit of compilation in a C++ program. Every C++ program is a collection of statements organized in such a way to achieve a goal. All the statements end with a ';'. Statements are executed in order from top to bottom when we run the program. Execution keeps going until there is a statement causing the program to terminate or run another sequence of statements. 
    //These are all examples of statements in a C++ program

  5. Input/Output: std::cout is used to display the output, whereas we use std::cin to take input from the user. 
    //Taking input

std::string name;
std::cout << "Enter your first name and age" << std::endl;
std::cin >> name; //Only one word string accepted
std::cin >> age;
    In order to accept Strings with blank spaces we use std::getline 
    //Data with Spaces
std::string full_name;
std::cout << "Enter your full name -" << std::endl;
std::getline(std::cin, full_name);

Variables and Data Types


  1. Number Systems: C++ provides support for the user to deal with different number systems by using suitable prefixes as shown in the follwoing code. 
    int dec_num = 15; //Decimal
int oct_num = 017; //Octal
int hex_num = 0x0f; //Hexadecimal
int bin_num = 0b00001111; //Binary
std::cout << "15 (Decimal) : " << dec_num << std::endl; //15
std::cout << "17 (Octal) : " << oct_num << std::endl; //15
std::cout << "0F (Hexadecimal) : " << hex_num << std::endl; //15
std::cout << "00001111 (Binary) : " << bin_num << std::endl; //15
  2. Integers and Initialization: Integers are stored using the data-type int. The maximum size of int is 4 bytes. An integer value can be initialized in various ways, some of which have been listed. 
    //Variable Braced Initialization
int garbage; //Stores a garbage value
int default {}; //0 is assigned by default
int a {7};
int b {3};
int sum {a+b}; //Using expressions to initialize.
int err {2.9}; //This will display an error or warning since int cannot store double values.
    //Functional Variable Initialization
int default (); //0 is assigned by default
int a (7);
int b (3);
int sum (a+b); //Using expressions to initialize.
int err (2.9); //Information will be lost as the value will be truncated to 2 and no warning will be displayed.
    //Assignment Initialization
int a=7;
int b=3;
int sum=a+b; //Using expressions to initialize.
int err=2.9; //Information will be lost as the value will be truncated to 2 and no warning will be displayed.
    We can also make use of the sizeof() function to print the size of a variable or data type. 
    std::cout << "Size of int is " << sizeof(int) << "bytes" << std::endl;
//Output - Size of int is 4 bytes
